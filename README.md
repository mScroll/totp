# TOTP アプリ  
RFC 6238 のワンタイム パスワードを生成するアプリです。  
  
・[Web App](https://www.mscroll.net/totp/webapp/)  
個人情報の取り扱いについては、公式サイトの「このサイトについて」に記載しております。  
  
ダウンロードおよび更新履歴は公式サイトよりご確認ください。
